# wishlist-api

express	Основний веб-фреймворк для створення маршрутів (routes), обробки HTTP-запитів та відповідей.
sequelize	ORM (Object-Relational Mapping), яка дозволяє працювати з базою даних MySQL як з об'єктами JavaScript, замість написання сирого SQL-коду.
mysql2	Драйвер (конектор), який дозволяє Node.js фізично підключатися до твоєї бази даних MySQL.
zod	Бібліотека для валідації вхідних даних. Вона перевіряє, чи прийшли правильні назви полів, типи даних (рядок/число) та довжина тексту перед тим, як зберегти їх у базу.
multer	Middleware для обробки завантаження файлів (multipart/form-data). Саме вона відповідає за збереження фото твоїх бажань у папку uploads.
dotenv	Дозволяє зберігати конфіденційні дані (пароль до бази, секретні ключі) у файлі .env, не показуючи їх у коді.
